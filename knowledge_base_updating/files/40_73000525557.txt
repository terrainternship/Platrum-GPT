
# Список задач

- Об инструменте

## Об инструменте
Вкладка «Задачи» - одна из составляющих инструмента “Задачи”, представляет собой список всех задач, доступных пользователю. Данная страница нужна, чтобы подвести какую-то отчетность по задачам или быстро найти нужные задачи по заданным параметрам.
https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/73003646286/original/N8OZvfyfccuRI2XAvqbFNnncAdJpxupvDQ.png?1646061196

## Кому доступен
Вкладка списка задач доступна всем сотрудникам. Сотрудник с базовым доступом может поставить задачу себе или любому другому сотруднику компании.
На странице с задачами сотрудник будет видеть задачи:
- поставленные им
- поставленные ему
- поставленные его подчиненным
Исключение из этой механики составляют Владелец проекта и сотрудники с доступом Администрирование всех задач.
https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/73003646284/original/jsJZCnfaRnBFNq5U6BNmaKoSyWOef0-FBA.png?1646061195
Такие сотрудники видят задачи всех и могут редактировать/удалять их, а также ставить задачу любому сотруднику..

## Фильтрация и сортировка
В списке задачи выстраиваются в порядке добавления, но можно задать свою сортировку по Статусу, Названию задачи, Продукту, Исполнителю и Постановщику, а также дате постановки и времени выполнения.
Фильтровать все задачи в списке можно по нескольким критериям:
- Содержит - можно указать слова из названия задачи, по которым будет выполнен поиск (например, все задачи, связанные со счетами)
