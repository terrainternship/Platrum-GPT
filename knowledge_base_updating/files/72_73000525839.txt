
# Настройки склада


СОДЕРЖАНИЕ
- Создание склада
- Создание категорий товаров
- Создание дополнительных полей
Начать работу со Складом необходимо с создания склада и добавления категорий товаров. Чтобы это сделать, нужно открыть Настройки, внизу  будет подраздел Склад.
Когда в проекте еще не добавлено ни одного склада, надо начать с создания первого.
https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/73003744830/original/QOuRvaXhRYIDelCgrv_85uFZzc4-_jdIBw.png?1646165502

### Создание склада
В разделе “Склады” нажмите кнопку “Создать” в правом верхнем углу.
Откроется меню добавления склада.
https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/73003744829/original/TPAP-a5AkZN5T8Zvxjd1KDcq2tYI23dtsQ.png?1646165501
В нем нужно прописать Название, а также отметить, считать ли его складом по умолчанию или нет. Для завершения нажмите “Сохранить”.
Созданный склад можно будет в любой момент отредактировать или удалить.
Вы можете создавать неограниченное количество складов в своем проекте, например, если у вас несколько офисов, в каждом из которых надо вести раздельный учет.

### Создание категорий товаров
После того, как склад создан, нужно создать категории товаров.
По кнопке Добавить категорию, где будет только одно поле для названия категории
https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/73003744825/original/VPq34RMoikrVj4vDiU_Y6cfGnJVb2HhS5w.png?1646165498
Справа от названия категории есть настройки процента амортизационных отчислений, о них отдельно будет описано в статье "Амортизация".
Любой категории можно добавить подкатегорию, нажав на +:
https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/73003744827/original/sE4ZvvO2IDC8N6J3j4KbfxGhuMSML-BpTw.png?1646165500
Каждой подкатегории можно задать свои настройки амортизации и создавать подкатегории.
Добавление товаров и привязку их к сотрудникам подробнее рассмотрим в статье
“Работа с товарами” (https://platrum-techsupport.freshdesk.com/ru-RU/support/solutions/articles/73000525831)
.

### Создание дополнительных полей
Вы можете добавить в карточку товаров дополнительные поля, чтобы внести больше информации об имуществе компании.
Для этого перейдите в настройки проекта, в разделе "Склад" нажмите на вкладку "Дополнительные поля". Здесь вы можете создавать, редактировать и удалять дополнительные поля.
Для добавления поля нажмите в правом верхнем углу на "Добавить поле":
https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/73037404768/original/VI3v-R_OakqDEijzIy50s0bTvv3jA93jzA.png?1672300866
Введите обязательно название и выберите тип поля:
https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/73037404773/original/i9iDENIfAt83tK9OnlppYkLavFnhHlJ6Ww.png?1672300881
Значение по умолчанию можно заполнить по желанию, оно будет автоматически подставляться при создании товара.
В дополнительных настройках можно указать сортировку, чтобы определить порядок полей. Обратите внимание, что все дополнительные поля всегда будут отображаться после полей по умолчанию.
Системное название заполняется автоматически, в проекте оно не отображается, но по желанию вы можете его изменить.
После добавления дополнительного поля вы сможете увидеть его в карточке товара:
https://s3.amazonaws.com/cdn.freshdesk.com/data/helpdesk/attachments/production/73037405465/original/41iDNiYMwdCew5vMHBcIbbWu0OuPtPx9bg.png?1672301576
